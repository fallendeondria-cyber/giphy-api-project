# Giphy API Project

This project uses the GIPHY API and Axios to allow users to search for and display GIFs.

## Features
- Search for GIFs using a keyword
- Append multiple GIFs to the page
- Clear all GIFs with a button

## Technologies Used
- HTML
- CSS
- JavaScript
- Axios
- GIPHY API

## How to Run
Open index.html in your browser.
